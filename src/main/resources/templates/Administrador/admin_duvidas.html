O que cada coisa faz?

Listar Pesquisar Adicionar Reservas

  • Laboratórios

    • Lista todos os laboratórios no sistema com as opções de alterar e excluir (a opção de excluir apaga todas as reservas criadas do laboratório)

  • Usuários

    • Lista todos os usuários do sistema com as opções de alterar e excluir (a opção de excluir apaga todas as reservas criadas pelo usuário)


  • Laboratórios

    • Lista os laboratórios a partir de um nome escolhido com as opções de alterar e excluir (alterar ou excluir um laboratório retornar uma lista com todos)

  • Usuários

    • Lista os usuários a partir de um nome escolhido com as opções de alterar e excluir (alterar ou excluir um usuário retornar uma lista com todos)

  • Reservas

    • Lista as reservas de um laboratório escolhido em um intervalo de tempo pré-selecionado (reservar ou cancelar uma reserva não cancela o filtro)


  • Laboratórios

    • Cria um novo laboratório no sistema (O laboratório criado não pode ser reservado ate que seja criado as reservas )

  • Usuários

    • Cria um novo usuário no sistema


  • Exibir

    • Exibi todas as reservas de todos os laboratórios de um mês selecionado (Essa opção pode demorar um pouco )

  • Disponibilizar

    • Cria as reservas para todos os laboratórios do sistema em um determinado semestre, selecionado o ano e depois o semestre. Laboratórios com as reservas já criadas são ignorados. Esta opção demora muito, é recomendado que deixe o sistema off-line enquanto processa.

  • Gráfico

    • Cria um gráfico com a quantidade de reservas em um mês para todos os laboratórios, exibindo a quantidade reservada e disponível.
